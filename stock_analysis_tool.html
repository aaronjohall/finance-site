Stock Analysis Tool

Input a stock ticker, retrieves financial data using yfinance, calculates basic ratios and outputs key insights.

What can be viewed with this tool?

  • Ticker Symbol e.g. (COST for Costco)
  • Long Name
  • Sector
  • Current Price
  • Market Cap
  • trailingPE
  • forwardPE
  • Price to Book
  • profitMargins
  • Return on Equity
  • beta
  • dividendYield
  • 52WeekChange

Example data as of 26/06/2025 - on VSCode

Why use this tool?

It will feedback an instant snapshot of all the indicators above from yfinance, using real-time data. Not sure what 'trailingPE' or 'beta' means? Click on the hyperlinks above or headover to the glossary page for a simple or an in-dept explanation on each term.
